# Morpheus Data Wrangling Benchmark

This dataset consists of 80 data preparation problems using the input-output examples provided by authors of Stackoverflow posts. 
Every problem has a single example.
Grammars are centered around `tidyr` and `dplyr`, two popular table manipulation libraries for R.

Benchmark description and grammars are implplemented following Morpheus (appendix of Arxiv paper: https://arxiv.org/pdf/1611.07502). 

See
> Yu Feng, Ruben Martins, Jacob Van Geffen, Isil Dillig, and Swarat Chaudhuri. 2017. Component-based synthesis of table consolidation and transformation tasks from examples. In Proc. Conference on Pro- gramming Language Design and Implementation. ACM, 422–436
